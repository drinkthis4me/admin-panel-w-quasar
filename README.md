# Admin Panel 

An admin panel build with Quasar.

![img](doc/Screenshot%202023-03-11%20at%2018-03-32%20Quasar%20App.png)

Tools used: 
- Quasar
- Vue-router
- Axios: HTTP client for calling backend API
- Pinia: State management

Backend: express + mysql2

## Project Setup

```sh
npm install
npm run dev
```
